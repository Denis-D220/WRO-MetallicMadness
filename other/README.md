# Other – Technical Documentation

This folder contains additional technical documentation required to understand, reproduce, and maintain the Metallic-Madness WRO 2026 vehicle system.

It includes:

- Hardware component datasheets
- Communication protocol diagrams
- Electrical references
- System documentation supporting the STM32 and Jetson architecture

This folder serves as the technical backbone of the project and ensures:

- Full hardware traceability
- Component verification
- Compliance with WRO documentation requirements
- Engineering transparency

Subfolders:

- `datasheet/` ? Official manufacturer documentation for all major components
- `flowchat/` ? Communication protocol flow diagrams and UART protocol definition

All documents stored here are official reference materials used during development.
